Project Description

This game was developed during a weekend Game Jam. It is a tower defense game featuring a scalable wave system, ensuring progressively increasing difficulty as the game continues. The game includes multiple tower types, each with distinct attack mechanics, providing strategic variety in how players defend against incoming enemies. The enemies themselves vary in attributes such as health, armor, and speed, requiring different strategies to counter them effectively.
