Project Description

This project is an Arkanoid‑style game developed in Unity as part of a university assignment. It is one of the first complete games I have created, aside from an earlier project made during a practices at Ten Square Games back in 2016. The gameplay follows the classic Arkanoid mechanics, where the player controls a paddle to bounce a ball and break bricks.

The game includes multiple levels with blocks of varying durability and point values. A simple user interface is implemented, providing essential elements such as a score counter and basic menus. The project serves as an introduction to game development, focusing on core gameplay mechanics, level progression, and UI integration within Unity.
