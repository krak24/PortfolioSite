Project Description

This visual novel is being developed during a Game Jam and is currently in progress. The project is powered by my custom command-based dialogue engine built in Unity, originally developed for the Glass Thoughts project. For this new title, the engine was significantly expanded to support additional features such as voice acting playback and full Live2D character support via Cubism SDK integration.

In addition to the story-driven narrative, the game features a cooking minigame that complements the overall plot and pacing. Once development is complete and the game is uploaded, I will include the relevant download links below.
