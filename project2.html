Project Description

This project showcases an advanced AI system for a horror game developed for the Themed Horror Game Jam #21. It is fully implemented in C++ within Unreal Engine, using native tools such as Behavior Trees and the Environment Query System (EQS).

The project focuses on designing and implementing two distinct enemy types with unique behavior patterns. These enemies utilize dynamic decision-making through Utility AI-style logic layered on top of Behavior Trees. My main responsibility is implementing AI behavior logic in C++, including perception handling, blackboard integration, and custom EQS queries. This is a collaborative project developed over a short timeframe for the jam, and will be released publicly after submission.

Development progress is being shared through our team’s LinkedIn development log. A link to the finished game will be added once the jam concludes and the project is submitted.
