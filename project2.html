Project Description

This project will present an advanced AI system, built on a Behavior Tree structure enhanced with Utility AI for dynamic decision-making. The system will manage combat through a Utility Combat Manager, analyze the environment using the Environment Query System (EQS), and utilize an advanced navigation system inspired by Death Stranding. The Behavior Tree structure is modeled after the AI design in Just Cause 3, enabling NPCs to react adaptively to changing game conditions. The entire system is being implemented in Unreal Engine, leveraging Blueprints and C++. The navigation system is designed for dynamic path recalculations, ensuring that NavMesh is built only around AI agents and the player, optimizing performance. Additionally, it incorporates a hierarchical path system (Highway System), allowing AI to navigate using predefined high-level paths when traditional NavMesh navigation fails. This approach enhances AI movement, especially in complex and open environments. This is an ongoing project that is currently in development.
