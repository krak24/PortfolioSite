Project Description

This project showcases an advanced AI system for a horror game developed for the Themed Horror Game Jam #21. It is fully implemented in C++ within Unreal Engine, combining Behavior Trees with Utility AI for adaptive decision-making.

The AI supports two distinct types of enemies, each with unique behavior patterns. The system includes a Utility-based Combat Manager, EQS-driven decision support, and a custom navigation system optimized for open-world traversal. The navigation logic draws inspiration from *Death Stranding* and *Just Cause 3*, incorporating features such as dynamic NavMesh bounds limited to agents and player positions, as well as a highway system for fallback hierarchical pathfinding.

This ongoing project is developed by a small team, and I am responsible for implementing AI behavior logic in C++. A full write-up will be posted after the jam ends. The game will be made available on Itch.io after submission. Development progress is documented in our public LinkedIn devlog by the lead programmer.
